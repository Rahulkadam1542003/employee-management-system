# Employee Management System

Full Stack web application for managing employees using Spring Boot and React.

### Tech Stack
**Backend:** Java, Spring Boot, Spring Security, MySQL, Maven  
**Frontend:** React.js, Axios, React Router, Bootstrap

### Features
- JWT Authentication with Login/Signup
- Role Based Access - Admin & Employee  
- CRUD Operations for Employees
- Responsive UI for all devices

### How to Run Locally
**Backend**
1. `cd backend`
2. Update `src/main/resources/application.properties` with your MySQL credentials
3. `mvn spring-boot:run`

**Frontend** 
1. `cd frontend`
2. `npm install`
3. `npm start`
